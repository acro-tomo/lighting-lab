# Share Demo Assets

These demo assets are original sample materials for LDK Lighting Lab.

- `dimensioned-ldk-demo-plan.svg`: original fictional dimensioned LDK plan.
- `share-demo-project.json`: importable LDK Lighting Lab project using that plan as the calibrated 2D background.

Copyright note: the floor plan is generated for this repository and is not copied from a real listing, builder catalog, or architectural drawing.
